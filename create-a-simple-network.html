Instructions

Part 1: Build a Simple Network

In this part, you will build a simple network by deploying and connecting the network devices in the Logical Workspace.

Step 1: Add network devices to the workspace.

In this step, you will add a PC, laptop, and a cable modem to the Logical Workspace.

A cable modem is a hardware device that allows communications with an Internet Service Provider (ISP). The coaxial cable from the ISP is connected to the cable modem, and an Ethernet cable from the local network is also connected. The cable modem converts the coaxial connection to an Ethernet connection.

Using the Device-Type Selection Box, add the following devices (category > sub-category > device):

  • PC: End Devices > End Devices > PC
  • Laptop: End Devices > End Devices > Laptop
  • Cable Modem: Network Devices > WAN Emulation > Cable Modem

Step 2: Change display names of the network devices.

  1. Click the device icon in the Logical Workspace.
  2. Click the Config tab in the device configuration window.
  3. Enter the new display names: PC, Laptop, and Cable Modem.

Step 3: Add the physical cabling between devices.

Using the Device-Type Selection Box, add the physical cabling between devices on the workspace.

  1. Connect PC FastEthernet0 to the wireless router Ethernet 1 using a copper straight-through cable.
  2. Connect the wireless router internet interface to the cable modem Port 1 using a copper straight-through cable.
  3. Connect the cable modem Port 0 to the internet cloud Coaxial 7 using a coaxial cable.

Part 2: Configure the End Devices and Verify Connectivity

You will connect a PC and a laptop to the Wireless router. The PC will use Ethernet; the Laptop will be converted to wireless by replacing its NIC. After both are connected, verify connectivity to cisco.srv.

Each device will obtain an IP address. IP addresses identify devices on a network and allow data transfer.

Step 1: Configure the PC.

Configure the PC for the wired network.

  1. Open the PC. In the Desktop tab, go to IP Configuration and verify DHCP is enabled and an IPv4 address is present.
  2. If no IPv4 address, select DHCP and observe the process.
  3. Open Command Prompt and enter ipconfig /all to review addressing (expect 192.168.0.x).
  4. Note: IPv4 uses dotted decimal (x.x.x.x). IPv6 was introduced to overcome IPv4 limits.
  5. Test connectivity: ping cisco.srv. Expect four replies.

Step 2: Configure the Laptop.

Configure the Laptop for wireless access.

  1. Open Laptop, select the Physical tab.
  2. Remove Ethernet copper module; install the wireless WPC300N module (power off first).
  1. Power off Laptop.
  2. Remove Ethernet module (drag to MODULES pane).
  3. Drag WPC300N module to empty slot.
  4. Power on Laptop.
  1. Open Desktop > PC Wireless.
  2. Open Connect tab. Select HomeNetwork (Refresh if needed) and click Connect.
  3. Open Web Browser and navigate to cisco.srv.
