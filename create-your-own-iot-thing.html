Packet Tracer - Create Your Own IoT Thing

Objectives

Part 1: Create Your Own Thing

Part 2: Save Your New Thing

Background / Scenario

In this activity, you will create a new IoT Thing (a Security Camera) and save it as a reusable template in Packet Tracer.

Instructions

Part 1: Create Your Own Thing

Create and configure a new IoT Thing, change its icon, and connect it to the wireless network.

Step 1: Add a new generic IoT device.

a. Open the Packet Tracer file.

b. Add a new device by selecting Thing from the Components area of the Device Selection box.

c. Drag the Thing icon to the workspace.

Step 2: Change the Display Name.

a. Click the Thing to open its configuration.

b. Click the Config tab and change the Display Name to Security Camera.

Step 3: Use the Thing Editor to change properties.

a. Click the Thing Editor tab (Properties subtab opens by default).

b. Change the Component Name to Security Camera.

c. Choose the Digital radio button, then set Slot Mapping to Slot 1.

Note: The next two substeps are done in this instructions file, not the embedded Packet Tracer instructions window.

d. Open this instructions file if it is not already open.

e. Right‚Äëclick the image below and select ‚ÄúSave image as‚Ä¶‚Äù. Store it for use as the device icon.

f. In Properties click New.

g. Choose the saved image and click Open. The icon updates.

Step 4: Add the Security Camera to the network.

a. In the device window, click I/O Config.

b. Set Network Adapter to PT-IOT-NM-1W-AC (Wi‚ÄëFi).

c. Wait for it to associate to the wireless network.

d. Click Config > Wireless0.

e. Confirm it received a 192.168.25.x IPv4 address (DHCP).

Step 5: Test connectivity.

a. On the Tablet open Desktop > Command Prompt.

b. Ping the Security Camera‚Äôs IPv4 address. Successful replies confirm reachability.

Part 2: Save the New Device

Save the configured device as a template for reuse.

Step 1: Save with Device Template Manager.

a. Tools > Custom Device Dialog (Device Template Manager opens).

b. Click Select, then click the Security Camera in the workspace.

c. The template name shows as Security Camera.

d. Check Home (placement category).

e. Click Add, then Save (accept template folder).

f. In the Device Selection box (Home), scroll to find your Security Camera.

Step 2: Verify template availability.

a. File > New (save current when prompted).

b. In the blank file, confirm the Security Camera appears under Home devices.

You created and connected a new IoT device and saved it as a template for future use.
