Packet Tracer - Edit Topologies

Objectives

In this activity, you will install a patch panel and a wall mount. You will then use these to connect network devices in the office to the equipment in the wiring closet.

Part 1: Add an Additional Switch to the Rack in the Office Network

Part 2: Connect a PC to the Office Network

Part 3: Work with Clusters

Part 4: Add a Second Home Cluster to the Network

Required Resources

Latest Packet Tracer version

Instructions

Part 1: Add an Additional Switch to the Rack in the Office Network

Step 1: Install and connect switch Office-SW2.

a. In the Wiring Closet, drag Office-SW2 from the Shelf to any open Rack space.

b. Select a copper straight-through cable from the Cable Pegboard.

c. Connect Office-SW2 FastEthernet0/1 to Patch Panel Jack1.

d. Select another copper straight-through cable.

e. Connect Office-SW2 GigabitEthernet0/1 to Office-SW1 GigabitEthernet1/1/1.

Part 2: Connect a PC to the Office Network

Step 1: Connect PC to wall mount.

a. Return to the Office (Alt + Left Arrow).

b. Add a straight-through cable from Office-User to the copper wall mount.

c. Click Connections (orange lightning bolt).

d. Select a copper straight-through cable.

e. Connect Copper Wall Mount1 Jack1 to Office-User FastEthernet0.

f. Select another copper straight-through cable.

g. Click Copper Wall Mount1 Punch Down1.

h. Connect to Patch Panel PunchDown1 (enter Equipment Cabinet > Rack).

i. Navigate: Rack > Patch Panel > PunchDown1.

Step 2: Test network connectivity from Office-User.

From Office-User, ping the Office-Admin PC (192.168.20.5).

a. Open Office-User > Desktop.

b. Open Command Prompt.

c. Ping the Office-Admin PC; replies should succeed.

C:\> ping 192.168.20.5

d. Open Web Browser and browse to office.srv.

If unsuccessful, verify cabling.

Part 3: Work with Clusters

Step 1: View the city in logical mode.

a. While in Office, click Back Level.

b. Switch to Logical workspace mode.

Step 2: Un-cluster and re-cluster an existing cluster.

a. Drag-select around the Library cluster.

b. Press Delete.

c. In confirmation, click Uncluster.

Note: If deleted instead, Undo (Ctrl + Z) or reset the activity.

d. Devices appear un-clustered; press Esc to return pointer.

Step 3: Re-cluster Library devices.

a. Drag-select all former Library devices.

b. Click Create New Cluster (Shift + U).

c. If a device was missed, use Move Object (Shift + M) > Move to cluster.

d. Rename cluster back to Library.

Part 4: Add a Second Home Cluster to the Network

Step 1: Add devices.

Add a PC, Home Gateway, and Cable Modem.

a. From End Devices, add PC.

b. From Network Devices: add Home Gateway (Wireless Devices) and Cable Modem (WAN Emulation).

Step 2: Connect devices.

a. Copper straight-through: PC0 FastEthernet0 to Home Gateway1 Ethernet1.

b. Copper straight-through: Home Gateway1 Internet to Cable Modem0 Port1.

Step 3: Cluster the devices.

a. Drag-select the three devices.

b. Click Create New Cluster.

c. Rename cluster to Second Home.

Step 4: Connect Second Home to the ISP.

a. Select blue Coaxial cable and click Second Home.

b. Choose Cable Modem0 Port0 (coaxial).

c. Click ISP cluster > Cloud0 Coaxial9.

d. Second Home cluster now connected to ISP.

Step 5: Note devices added to Second Home.

a. Use Place Note tool; click near cluster.

b. List added devices (PC, Home Gateway, Cable Modem).
