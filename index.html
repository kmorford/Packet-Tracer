Packet Tracer — Fall 2025

Networking Labs, Templates, and IoT Scenarios

Quick Start

  1. Install Cisco Packet Tracer (latest version).
  2. Clone or download this repo:
    git clone https://github.com/kmorford/Packet-Tracer.git
  3. Open .pkt files and follow the embedded instructions.

Repository Map

  • Cisco Packet Tracer — core labs & device configs
  • Networking — VLANs, routing, services
  • IoT — sensors, actuators, and event-driven labs

Learning Outcomes

  • Design & configure small to medium networks in Packet Tracer.
  • Implement IP addressing plans (VLSM), VLANs, trunking, and routing.
  • Configure routing protocols (RIPv2, OSPF).
  • Deploy services (DHCP, DNS) and validate connectivity.
  • Build and test simple IoT topologies.

Submission Guidelines

Unless directed otherwise, submit:

  1. Your completed .pkt file.
  2. A short write-up (.pdf or .md) including:
    • IP/VLAN plan table
    • Config snippets (show run, show ip route, etc.)
    • Verification output (ping, tracert)
    • 2–3 problems encountered + fixes

File naming: LastName_FirstName_LabNN

Troubleshooting Checklist

  • Are interfaces up? (show ip int br)
  • Correct VLANs and trunking?
  • Default gateway set?
  • Routing protocols advertising networks?
  • Any ACL/security blocking traffic?
  • No IP overlaps?
