Packet Tracer - Examine Packets in the Small Office

Objectives

Part 1: Create a Simple PDU in Simulation Mode

Part 2: View Contents of PDUs

Part 3: Create a Complex PDU in Simulation Mode

Background / Scenario

In this Packet Tracer activity, you will use Simulation mode to create PDUs to explore network functionality.

Note: Simulation mode does not work in Physical mode; therefore, Physical mode is locked.

Instructions

Part 1: Create a Simple PDU in Simulation Mode

Step 1: Enter the Office network.

Click Office to view the logical network for the Office.

Step 2: Enter Simulation mode.

Click the Simulation mode icon in the bottom-right corner. This pauses all network traffic that occurs in Realtime mode.

Note: If an ‚ÄúIntercity is locked‚Äù pop-up appears, click Okay.

Step 3: Create a simple PDU (ping Office-Admin to Printer0).

a. Click Add Simple PDU (closed envelope icon).

b. Click Office-Admin (source) then Printer0 (destination).

Step 4: Observe traffic.

a. In Play Controls, click Capture/Forward.

b. Click repeatedly to advance packet hops; watch the Event List.

c. Continue until the return ICMP reaches Office-Admin. If ‚ÄúBuffer Full‚Äù appears, you have many events.

d. When Buffer Full appears, click View Previous Events. If you clicked Clear Event List, use Reset Simulation and repeat.

Part 2: View Contents of PDUs

Step 1: Use the Event List.

Inspect the first ICMP PDU from the PC.

a. In Event List, widen the panel if needed.

b. Click the first ICMP entry (Type column) to open PDU Information at Device: Office-Admin.

c. Review the OSI Model tab (outbound Layer 3, source/destination IPv4).

d. Click Outbound PDU Details for protocol headers.

e. Close the window.

f. Explore other PDUs similarly.

Step 2: Delete the simple PDU.

Double-click Delete in the Event Simulation pane to remove the simple PDU and clear events.

Part 3: Create a Complex PDU in Simulation Mode

Step 1: Add the complex PDU (Office-Admin to Printer0).

a. Click Add Complex PDU (open envelope).

b. Click Office-Admin (source) ‚Äî the Create Complex PDU window opens.

c. Click Printer0 (destination) ‚Äî destination IP fills in.

Step 2: Configure periodic pings (5 s).

a. Confirm PING application selected.

b. Enter 1234 in Sequence Number.

c. Select Periodic, set Interval to 5 seconds.

d. Click Create PDU (first ICMP appears in Event List).

Step 3: Observe periodic traffic.

a. Click Play; watch new PDUs every 5 seconds.

b. Adjust animation speed with the slider.

c. Click Play again to pause simulation.

d. To delete, double-click Delete in Event Simulation pane.
